# AI Student Performance Analysis

## Overview

This project analyzes how Generative AI usage influences academic performance, learning retention, and student well-being among university students. Using Python and Power BI, the analysis explores behavioral patterns, academic outcomes, and institutional policy impacts to provide data-driven recommendations for higher education.

## Business Objectives

- Analyze the relationship between AI usage and academic performance.
- Evaluate student well-being indicators such as burnout and anxiety.
- Measure learning retention across AI usage segments.
- Compare institutional AI policies and their impact.
- Develop interactive dashboards to support decision-making.

## Key Findings

- Average GPA increased by approximately **6.35%** after one semester of AI usage. :contentReference[oaicite:0]{index=0}
- Weekly AI usage showed **almost no linear correlation** with academic performance (Pearson correlation = -0.02). :contentReference[oaicite:1]{index=1}
- Institutions with **Strict Ban** policies recorded the highest average anxiety levels. :contentReference[oaicite:2]{index=2}
- AI usage was relatively consistent across all academic levels. :contentReference[oaicite:3]{index=3}
- Moderate AI users achieved the highest Skill Retention Score. :contentReference[oaicite:4]{index=4}

## Recommendations

- Establish responsible AI usage guidelines instead of complete restrictions.
- Improve AI literacy through prompt engineering and AI ethics training.
- Combine AI-assisted learning with critical thinking activities.
- Monitor student well-being using burnout and anxiety indicators.
- Utilize interactive dashboards for continuous academic monitoring. :contentReference[oaicite:5]{index=5} :contentReference[oaicite:6]{index=6}
